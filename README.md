# Smart Home Project

## Description
Ce projet présente un système Smart Home basé sur les technologies IoT.
Il s'agit d'une simulation académique utilisant la méthode Agile.

## Objectif
Organiser le projet avec Product Backlog, Sprints et User Stories.
